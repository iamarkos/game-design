# Game Design & Prototypes Archive

A consolidated archive of early game development prototypes, microgames, and physical game design documents. These projects showcase a foundational approach to player experience, puzzle logic, and object-oriented programming.

## 🕹️ Included Projects

### 1. Top Island (Java GUI Game)
An interactive desktop game where players must guess which island has the greatest average height. 
*   **Tech Stack:** Java (Object-Oriented Programming)
*   **Features:** Dynamic graphical user interface (GUI) featuring a countdown timer, scoring system, and integrated sound and visual effects. 
*   *See the repository files for the source code and gameplay screenshots.*

### 2. Backstage (Bitsy Microgame)
A browser-based microgame focused on game-based learning, placing the player in the role of a theater stage manager.
*   **Tech Stack:** HTML / Bitsy
*   **Concept:** Simulates behind-the-scenes theater life, requiring the player to complete a series of management tasks to ensure everything is set before the curtain rises. 
*   *The game is fully contained within the provided HTML file and can be played directly in the browser.*

### 3. Egyptians’ Last Puzzles (Escape Room Design)
Comprehensive design documentation for a physical, themed escape room experience. 
*   **Format:** Design Documents (PDF)
*   **Concept:** Players must solve narrative-driven puzzles, navigate time-limit events, and crack mind-bending riddles to unlock temple gems.
*   **Included Files:** Level Design layout, Narrative Documentation, One-Sheet overview, and a Vertical Slice Design Document.

### 4. Spiral Rivals (Print-and-Play Board Game)
A highly constrained, two-player physical board game designed to be played using only a single sheet of A4 paper.
*   **Format:** Rules & Game Board (PDF)
*   **Concept:** Showcases minimalist game design and rule-setting. One half of the page contains the complete mechanics and rules, while the other serves as the playable board.
